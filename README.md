# product-preview-card
A Front-End design for a product preview card component and pricing


**Description** </br>
Just a responsive product preview card  with component  which is a responsive  UI element used to
display key product details—image, category, name, description, price, and a call-
to-action button—in a compact, card-like format. It is commonly used in e-commerce
for layout, often featuring, for example, perfume details.  </br>
